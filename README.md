
# MergX v8.55 — Dark Foundation (Next.js App Router)

## Snabbstart
```bash
npm i
npm run dev
# http://localhost:3000
```

## Moduler
- Dashboard (sök, 4 KPI, ekonomi-chart, AI‑rail)
- CRM (lista + insikter)
- Ekonomi (fakturor, utgifter, OCR-förslag)
- AI-Nav (coach, what‑if, alerts)
- AI‑Karta (mock-rutter, notiser)
- Chatt (filer/röststubbar)
- Inställningar (företag, villkor, AI-policy)
